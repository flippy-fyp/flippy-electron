# Flippy Qualitative Testbench

Score-follower qualitative testbench.

## Development Setup

- Install dependencies

```bash
npm install
```

- Run these two commands in separate consoles.

```bash
npm run dev:react
npm run dev:electron
```

- Run tests
```bash
npm test
```

- Lint
```bash
npm run lint
```

## Building

_Currently only tested on Windows._

```bash
npm run build
npm run dist
```
